# SDK Reference

The Lit JS SDK provides a comprehensive toolkit for integrating Lit Protocol's decentralised authentication and cryptographic capabilities into your applications. This reference documentation covers all the methods, classes, and configuration options available in the SDK.

### 1. Client Setup Layer
- **[Lit Client Setup](/sdk-reference/client-setup/lit-client)**: Configure the Lit Protocol client
- **[Auth Manager Setup](/sdk-reference/client-setup/auth-manager)**: Set up authentication management with configurable storage options
- **[Storage Options](/sdk-reference/client-setup/storage-options)**: Configure how authentication data is stored and managed
- **[Payment Setup](/sdk-reference/client-setup/payment-setup)**: Configure payment processing capabilities

### 2. Authentication Options
- **[EOA Authentication](/sdk-reference/auth/eoa-auth)**: Externally Owned Account authentication
- **[PKP Native Authentication](/sdk-reference/auth/pkp-native-auth)**: Programmable Key Pair native authentication options
- **[PKP Custom Authentication](/sdk-reference/auth/pkp-custom-auth)**: Custom PKP authentication implementations

### 3. Auth Context Management
- **[Context Generation](/sdk-reference/auth-context/generation)**: Generate authentication contexts
- **[Context Consumption](/sdk-reference/auth-context/consumption)**: Execute operations using authentication contexts

## Getting Started

If you're new to the Lit JS SDK, we recommend starting with our [Quickstart Guide](/quickstart) to get up and running quickly. Once you're familiar with the basics, this reference documentation will help you dive deeper into specific functionality.
